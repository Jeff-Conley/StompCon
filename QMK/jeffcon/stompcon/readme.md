# stompcon
![stompcon](https://github.com/Jeff-Conley/StompCon/blob/main/images/StompCon_Wiring_Diagram.png?raw=true)

A foot controlled 6 key macropad.

* Keyboard Maintainer: [Jeff Conley](https://github.com/Jeff-Conley)
* Hardware Supported: Raspberry pico 1 (RP2040)
* Hardware Availability: Build guide is located here: https://github.com/Jeff-Conley/StompCon

Make example for this keyboard (after setting up your build environment):
 
 
    make handwired/jeffcon/stompcon:default
    
    
flash the Raspberry Pi Pico by holding the bootsel button and plugging in the usb to enter booloader mode. Drag and drop the binary file into the RPI-RP2 drive that is added when plugging in.
